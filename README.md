The Car Rental System is a C++-based application developed to automate and manage the core operations of a vehicle rental service. It provides a centralized and structured solution for handling customers, vehicles, bookings, and financial transactions. By applying Object-Oriented Programming (OOP) principles such as encapsulation, abstraction, and polymorphism, the system ensures a modular, scalable, and maintainable design.

The system minimizes manual effort and improves accuracy by digitizing processes like vehicle tracking, billing, and user management. It is designed to simulate real-world rental business operations while maintaining data security through controlled access and well-defined user roles.

---

## Features

### 👤 User Management
The system allows customers to register, log in, and manage their personal profiles. It securely stores user data and ensures that each user can access and update their own information when needed.

### 🔐 Access Control
Role-based access control is implemented to restrict system functionality based on user roles such as admin, staff, and customer. This ensures that sensitive operations are only performed by authorized individuals.

### 🚘 Vehicle Inventory Management
All vehicles are stored and managed within a centralized inventory system. The system tracks availability, categories, and specifications of each car, and updates their status automatically when they are rented or returned.

### ⚙️ Car Feature Selection
Users can choose vehicles based on specific preferences such as air conditioning, fuel type, and seating capacity, allowing a more personalized rental experience.

### 💰 Payment Calculation
The system automatically calculates rental costs based on factors like duration, vehicle type, and selected features. This eliminates manual calculation errors and ensures consistency.

### 🧾 Invoice Generation
Detailed invoices are generated for each transaction, including rental duration, selected vehicle, additional services, taxes, and total payable amount.

### 🎁 Discount Management
The system supports promotional, seasonal, and loyalty-based discounts. These are applied automatically to eligible bookings to enhance customer satisfaction.

### 🔧 Maintenance Tracking
Vehicles that require servicing are flagged by the system. Such vehicles are temporarily restricted from booking to ensure safety and proper maintenance.

### ⏱️ Delay Management
Late returns are monitored, and penalties are calculated automatically. This helps maintain scheduling accuracy and enforces accountability among users.
